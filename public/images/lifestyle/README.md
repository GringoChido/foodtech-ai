# Lifestyle Photos

Drop your photos here, named by slot ID.

Example: `hero-bg-left.webp`, `founder-portrait.webp`

Then add the `src` path to the corresponding slot in `lib/image-config.ts`:

```ts
src: "/images/lifestyle/hero-bg-left.webp"
```

See `PHOTO-BRIEF.md` in the project root for the full shot list with directions.
